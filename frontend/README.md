# Frontend

Ziel: Web-App / Mobile PWA für Baustellenaufnahme, Planviewer, Galerie, Filter, Kommentare und Admin-Ansichten.

Status: Platzhalter. Die konkrete Frontend-Entscheidung zwischen Vue 3, React oder Inertia ist noch offen und muss vor Produktivcode in `docs/06_entscheidungen.md` dokumentiert werden.

UI-Grundsatz:

- mobile-first für Aufnahmefluss.
- dichte Desktop-Ansicht für Planviewer und Galerie.
- keine Marketing-Landingpage.
- keine Berechtigungslogik nur im Frontend.
