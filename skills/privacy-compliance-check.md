# Privacy Compliance Check

## Ziel

Features datenschutzfreundlich prüfen.

## Prüfen

- Welche personenbezogenen Daten?
- Warum werden sie verarbeitet?
- Welche Rechtsgrundlage?
- Wie lange gespeichert?
- Wie gelöscht?
- Wer hat Zugriff?
- Externe Anbieter?
- EU-/Drittlandübermittlung?
- Consent nötig?
- Datenschutzhinweis nötig?
- Export/Auskunft nötig?
- Sind Fotos, Pläne, Kommentare oder Audit-Logs betroffen?
- Wird KI, Voice, RAG oder Transkription verwendet?
