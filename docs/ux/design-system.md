# Design System

## Ziel

Die UI ist ein ruhiges, dichtes Arbeitswerkzeug für Baustelle, Bauleitung und Projektverwaltung.

## Farben

- Hintergrund dunkel: `#0f1720`
- Panel: `#151f2b`
- Panel 2: `#1c2938`
- Text hell: `#f3f6f9`
- Sekundärtext: `#aab7c4`
- Linie: `#2c3b4d`
- Primärakzent: `#41b6c4`
- Sekundärakzent: `#7c8cff`
- Erfolg: `#5dd39e`
- Warnung: `#f5c768`
- Fehler: `#ff7b7b`

## Typografie

- System UI oder freigegebene Unternehmensschrift.
- Mobile Eingaben und Fließtext mindestens 16 px.
- Keine unnötigen Uppercase-Stile.

## Komponenten

- Buttons: Primary, Secondary, Danger, Disabled.
- Formulare mit sichtbaren Labels.
- Tabellen mit responsive Alternative.
- Filter mit sichtbarem Status.
- Planviewer mit sichtbarer Planversion.
- Galerie mit Datum, Ort, Gewerk und Berechtigungsstatus.

## Layout

- Mobile-first für Aufnahme.
- Desktop für Planviewer, Galerie und Admin.
- Keine horizontalen Scrollflächen.
- Klare, wiederholbare Abstände.

## Assets

- Nur freigegebene Logos.
- Icons konsistent verwenden.
- Bilder komprimieren.
- Alt-Texte verwenden.
